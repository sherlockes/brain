1. Info

Homepage es un panel de inicio y dashboard altamente personalizable para tu infraestructura y servicios auto-alojados. Es una aplicación web rápida, segura y totalmente estática, configurable mediante archivos YAML o etiquetas Docker, y con soporte para más de 100 integraciones de servicios y widgets como tiempo, reloj y estado de contenedores. Permite agrupar enlaces y servicios en un solo lugar, mostrar estado y métricas, y extenderse con widgets personalizados. Su integración con Docker y Kubernetes facilita la detección automática de servicios y su monitorización. Está diseñado para ser usado detrás de un proxy seguro o VPN.

4. Variables de entorno

  • HOMEPAGEALLOWEDHOSTS: Separados por comas los host en los que está corriendo

5. docker-compose.yml

services:
  homepage:
    user: "1000:1000"
    image: ghcr.io/gethomepage/homepage:latest
    container_name: homepage
    ports:
      - 4000:3000
    volumes:
      - /home/sherlockes/dockers/homepage/data:/app/config
      - /home/sherlockes/dockers/homepage/data/icons:/app/public/icons
      - /var/run/docker.sock:/var/run/docker.sock # (optional) For docker integrations
    restart: unless-stopped
    environment:
      HOMEPAGE_ALLOWED_HOSTS: homepage.zgz.sherblog.es,192.168.10.211:4000
networks: {}

6. Gestión

docker compose up -d
