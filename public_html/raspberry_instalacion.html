Partimos con la raspberry vacía y estos son los pasos que hay que dar. He probado a hacer un script que con el paso del tiempo siempre falla en un punto u otro, así quedan las instrucciones claras.

1. Instalando PI OS Lite

1.1. Grabar la imagen mediante Imager y actualizar el sistema.

sudo apt update
sudo apt upgrade

1.2. Generar la llave ssh

ssh-keygen

1.3. Copiar la llave ssh a Github y GitLab.

Abrir el archivo ".ssh/idrsa.pub" con un editor de texto y copiar el contenido.

1.4. Instalar y configurar git

sudo apt install git
git config --global user.email "sherlockes@yahoo.es"
git config --global user.name "Sherlockes"

1.5. Clonar SherloScripts

git clone git@github.com:sherlockes/SherloScripts.git

1.6. Open Network Manager UI with para asignar ip estática

sudo nmtui

1.7. Instalar Pi-Hole

curl -sSL https://install.pi-hole.net | bash

1.8. Instalar Rclone (Web).

sudo -v ; curl https://rclone.org/install.sh | sudo bash
