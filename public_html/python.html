4. Creación de entornos virtuales

  • Instalar `virtualenv`:
sudo apt install python3-pip
sudo apt install python3-virtualenv
  • Crear un nuevo entorno virtual

Ve al directorio en el que deseas crear el entorno virtual y ejecuta el siguiente comando para crear uno nuevo. Puedes reemplazar `mientorno` con el nombre que prefieras para tu entorno virtual. 

virtualenv mi_entorno
  • Activar entorno virtual
source mi_entorno/bin/activate
  • Desactivar el entorno virtual
deactivate
