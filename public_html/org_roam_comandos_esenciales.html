org-roam comandos esenciales

Además de los utilizados en emacs, este es un resumen con los comandos esenciales de org-roam que uso habitualmente.

  • Buffer Org-Roam, Mostrar (org-roam-nbuffer-toggle) -> C-c n l
  • Enlace, Insertar (org-roam-node-insert) -> C-c n i
  • Enlace, Manual -> [[]] M-x completion-at-point
  • Nodo, Buscar o crear (org-roam-node-find) -> C-c n f
  • Nodo, Crear a partir de encabezado (org-id-get-create)
  • Nodo, Crear un alias (org-roam-alias-add)
  • Nodo, Anadir un tag (org-roam-tag-add)
