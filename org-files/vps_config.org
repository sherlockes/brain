* Configuración inicial
** Crear un nuevo usuario
~adduser sherlockes~
** Añadirlo al grupo sudo para que pueda administrar el sistema
~usermod -aG sudo sherbo~
** Copiar la llave ssh del pc local al vps (Desde la consola del pc local)
~ssh-copy-id sherbo@IP_DE_TU_VPS~
** Deshabilitar el login de "root"
~sudo nano /etc/ssh/sshd_config~
Editamos, guardamos la línea "PermitRootLogin no" y reiniciamos
~sudo systemctl restart ssh~
** Instalar Fail2Ban
~sudo apt update && sudo apt install fail2ban -y~
** Instalar Docker
#+begin_src bash
# 1. Instalar dependencias previas
sudo apt update
sudo apt install ca-certificates curl gnupg -y

# 2. Añadir la llave GPG oficial de Docker
sudo install -m 0755 -d /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
sudo chmod a+r /etc/apt/keyrings/docker.gpg

# 3. Configurar el repositorio oficial
echo \
  "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu \
  $(. /etc/os-release && echo "$VERSION_CODENAME") stable" | \
  sudo tee /etc/apt/etc/apt/sources.list.distrib > /dev/null

# El comando anterior a veces requiere apuntar directamente a sources.list en Ubuntu de Contabo:
echo \
  "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu \
  $(. /etc/os-release && echo "$VERSION_CODENAME") stable" | \
  sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

# 4. Instalar Docker y Docker Compose
sudo apt update
sudo apt install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin -y
#+end_src
